**D.Com Homepage**를 새로 만들 능력자분들을 구합니다! 구현하고자 하는 기능은 다음과 같습니다.
- 공지사항 : 동아리 공지사항을 올림
- 스터디 게시판 : 스터디 모집 및 스터디 진행 상황 보고
- 족보/솔루션 게시판 : 전에 있던 자료들을 옮길 예정
- 동아리 행사 게시판 : 동아리 행사 진행시 사진들을 올림
- 정보공유 게시판 : 블로그처럼 동아리 구성원이 정보 공유 관련 게시글을 올림
- 동아리 소개
